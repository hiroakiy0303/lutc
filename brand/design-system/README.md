# Design System

LUTC ConsultingのWeb表現は `swiss-design-system.json` を基準にします。

## 主要トークン

- Alpine Red: `#dc2626`
- Jet Black: `#09090b`
- Charcoal Gray: `#18181b`
- Zinc Gray: `#27272a`
- Structure Gray: `#71717a`
- Swiss Plaster: `#f4f4f5`
- System Borders: `#e4e4e7`

## 実装上の注意

- 角丸、影、グラデーションの多用は避ける。
- 余白と罫線でセクションの意味を分ける。
- 本文は日本語中心、見出しやセクション名に英語を効果的に使う。
- 紫、ネオン、未来都市、ロボットなどのAIらしすぎる表現は避ける。
