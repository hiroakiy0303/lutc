# Brand

LUTC Consultingのブランド表現を管理します。

## 採用中の正式ロゴ表現

- 表記: `LUTC`
- 形状: 横長の矩形ワードマーク
- 色: Alpine Red `#dc2626`
- 文字色: White `#ffffff`
- 用途: ヘッダー、フッター、faviconやOGP素材の基礎表現

## デザインシステム

`design-system/swiss-design-system.json` をブランドUIの基準とします。

基本方針:

- Swiss Styleをベースにした明快なグリッド
- 白、黒、グレー、Alpine Redに限定した抑制的な色設計
- 装飾ではなく、余白・罫線・タイポグラフィで構造を作る
- AIツール感よりも、経営コンサルティングとしての信頼性を優先する
